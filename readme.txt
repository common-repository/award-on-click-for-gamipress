== Description ==

This GamiPress Add-on adds a shortcode to show a link. The user is then awarded a specified achievement when the link is clicked. This shortcode is fully integrated with the GamiPress shortcode insert button.

**Note:** You need to install the free [GamiPress plugin](https://wordpress.org/plugins/gamipress/ "GamiPress")&trade; (version 1.4 or higher) to use the GamiPress Community Add-on. The Plugin was tested up to GamiPress 1.9.1

[Get the GamiPress plugin](https://wordpress.org/plugins/gamipress/ "GamiPress").

This plugin was originally developed for the [Academy of International Cooperation](https://www.giz.de/akademie/en/html/index.html) of [GIZ](https://www.giz.de).

== Installation ==

1. Upload, activate and configure the free [GamiPress plugin](https://wordpress.org/plugins/gamipress/ "GamiPress") to WordPress.
1. Upload 'gamipress-award-on-click' Add-on to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
